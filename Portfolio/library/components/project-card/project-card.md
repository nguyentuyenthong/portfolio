# Project Card

Card dự án trong portfolio grid — 16:10 media, tags, award line, hover lift.

## Usage

```html
<a class="uu-project-card" href="...">
  <div class="uu-project-card__media">...</div>
  <div class="uu-project-card__body">...</div>
</a>
```

Grid wrapper: `.uu-project-grid` (1 col mobile, 2 col ≥768px).

## Compose với

- `badge` — award highlight
- `tag` — category labels
